Player.Ads

Pixellot web-sdk's ads manager.

Works with preroll and midrolls which were passed to player instance on setSource.

Methods

# isAdsAllowed() → {boolean}

Returns Boolean value which represents possibility to play ads right now.

boolean
Example
const isAdsAllowed = player.ads.isAdsAllowed();

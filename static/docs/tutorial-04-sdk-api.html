SDK API

In order if you want to use SDK API you need to set up an authentication.

Authentication

When you got user's token and signature, the process of authentication is simple and straightforward.

In cases if you want to use SDK api methods you need to set projectID.

You can get projectID from partnerAPI

You can find more info here

const projectId = '00x00xxxx000xx00x0';
const token = '000XXXX0X0X0X0000X0X00X00X0X0X00X00X00X0X0XXXXX000';
const signature = 'piug*f&^r&R7T8y*y*ty&*t&^duvuf&^t(hpjp:m:kgrf&dcycugoy'
const { SDKApi } = window['pixellot-web-sdk'];

SDKApi.auth.setSession(token, signature);
SDKApi.auth.setProjectID(projectId);

In order to reset session:

SDKApi.auth.invalidateSession();

You can find more info here Usage:

const options = { streamName: 'hd', mode: 'sync', videoType: 'live' };

SDKApi.createTag(options)

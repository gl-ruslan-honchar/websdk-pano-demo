TagPlugin

TagPlugin(player, options)

Constructor

# new TagPlugin(player, options)

TagPlugin slightly changes UI of the player.

When registered it can show or remove passed tags from UI.

Parameters:
Name Type Description
player object

instance of player
options object

object of options to be passed inside of plugin
Properties:
Name Type Attributes Default Description
pluginName string <optional>
 'tag'

name of plugin to be used in player.plugins[name]. Should be a 'static' class property. Required.
exposedProps Array.<string>

array of props to be exposed on player instance
Example
// Extract and use as
const { PlayerTagPlugin } = window['pixellot-web-sdk'].Plugins;

player.registerPlugin(PlayerTagPlugin);

player.showTags(tags); // show passed tags on UI.

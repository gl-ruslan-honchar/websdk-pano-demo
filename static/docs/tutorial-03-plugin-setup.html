Plugins Setup

You can expand player functionality with attaching plugins to it. The're lots of possibilities you can achieve.

You can create your own custom plugin, or use existing ones, or extend existing and register your custom version.

Check out next steps.

Register plugin

Registration and usage of single plugin.

More info you can find here

const { TimeMovePlugin } = window['pixellot-web-sdk'].Plugins;

player.registerPlugin(TimeMovePlugin, { /* options */ });
// player.registerPlugin(TimeMovePlugin, { /* options */ });


// Than use it like
player.updateCurrentTime(5); // updates current time on 5 points.
// or: player.plugins.timeMove.updateCurrentTime(5); // updates current time on 5 points.

Register multiple plugins

Registration and usage of several plugins.

More info you can find here

const {TimeMovePlugin, TagCreatePlugin} = window['pixellot-web-sdk'].Plugins;

const myPlugins = [
    TimeMovePlugin,
    [TagCreatePlugin, { /* options */ }]
];

player.registerPlugins(myPlugins);

Extend plugin

Here`s how you can extend existing plugin.

const { TimeMovePlugin } = window['pixellot-web-sdk'].Plugins;

class CustomTimeMovePlugin extends TimeMovePlugin {
    constructor(player, options) {
        super(player, options)

        this.exposedProps.push('customTimeUpdate')
        //...
    }

    customTimeUpdate(time) {
        this.player.currentTime = time
    }
}

player.registerPlugin(CustomTimeMovePlugin);

// Than use it like
player.customTimeUpdate(5); // updates current time on 5 points.
// or: player.plugins.timeMove.customTimeUpdate(5);

Create a custom plugin

Also you can create your own plugin.

const { Plugin } = window['pixellot-web-sdk'];

class CustomPlugin extends Plugin {
    static pluginName = 'testPlugin'

    exposedProps = ['test']

    constructor(player, options) {
        super(player, options)
        //
    }
    //

    test () {
        console.log(123)
        // ...
    }
}

player.registerPlugin(CustomPlugin);
// or: player.registerPlugin(CustomPlugin, { /\* options /\* });

player.test() // 123
// or: player.plugins.testPlugin.test() // 123

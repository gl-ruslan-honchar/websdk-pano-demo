Tutorial

Introduction

Pixellot WebSDK allows you to create a video player with some expanded functionality.

Player have support of all main video playbacks including HLS video files.

Check out Tutorials and API sections.

Quick start

Here is our quick start with the WebSDK.

1. Include pixellot web-sdk script.

<script src="https://pixellot-web-sdk.pixellot.tv/v-{version}/pixellot-web-sdk.js"></script>
  • In case you want to use Player Ads Manager, following script is required.
<script type="text/javascript" src="//imasdk.googleapis.com/js/sdkloader/ima3.js"></script>

2. Player setup.

HTML:

 <div id="player"></div>

Script:

const container = document.getElementById('player');
const WebSDK = window['pixellot-web-sdk'];

const projectId = '00x00xxxx000xx00x0';
const player = WebSDK.Player(container); // or with `new` keyword

// Important to set your project id at the beggining.
// Later it's used for analytics & also required when you do api calls using SDKApi. 
WebSDK.SDKApi.auth.setProjectID(projectId);

player.setSource('/public/video-sample.mp4');
player.play();

3. Plugins setup (optional)

Check out plugins registration and plugins section

const { TimeMovePlugin, ClipCreatePlugin } = window['pixellot-web-sdk'].Plugins;
const plugins = [
    ClipCreatePlugin,
    [TimeMovePlugin, { /* options */ }]
]

player.registerPlugins(plugins);

// player.setSource('/public/video-sample.mp4');
// player.play();

player.startRecording();
// or player.plugins.clipCreate.startRecording(); // but not recommended!

setTimeout(() => {
    player.stopRecording();
}, 5000);

// New clip is created after 5 sec.

# registerManager(managerData)

Register manager on player.
Parameters:
Name Type Description
managerData object data with manager details
name string name of manager property in player object.
model Class class of manager which instance will be attached to player with followed options.
options object Additional parameters that passes to manager during creation.

# registerManagers(managers)

Register multiple managers on player.
Parameters:
Name Type Description
managers Array.<Class> array of managers to register on player.

# registerPlugin(PlayerPlugin, optionsopt, commonOptionsopt)

Register plugin on player.
Parameters:
Name Type Attributes Description
PlayerPlugin Class class of plugin which instance will be attached to player with exposed options. Should extend Plugin.
options object <optional>
 Additional parameters that passes to plugin during creation.
commonOptions object <optional>
 Additional parameters that passes to plugin during creation.(usually passed from player or when registered multiple plugins)
Example
class CustomPlugin extends Plugin {
    constructor(player, options) {
        super(player, options)
        //
    }
    //
}

player.manager.registerPlugin(CustomPlugin);
player.manager.registerPlugin(CustomPlugin, { /\* options /\* });

# registerPlugins(plugins, options)

Register multiple plugins on player.
Parameters:
Name Type Description
plugins Array.<Class> array of plugins to register on player.
options object options object
Example
// class CustomPlugin extends Plugin

player.manager.registerPlugins([CustomPlugin]);
player.manager.registerPlugin([CustomPlugin, [CustomPlugin2, { /\* options /\* }] ]);
